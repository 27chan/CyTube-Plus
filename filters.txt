Formatted codes for CyTube Plus filters.

Use if you have problems with installing with special buttons, or if you want to do it manually.

1. Go to "Moderation Menu" and click "Chat Filters".
2. Scroll down to "Multi-Filter Editor".
   WARNING! You should see "Install Fonts" and "Install Emotes" buttons.
   ^ I recommend you just click them to prepare the code automatically.
   ^ If you don't see those buttons, you must paste the code manually.
3. Copy appropriate code from below (be careful and do not include comments!).
4. Paste the code.
5. Click "Add/Update" button.

Note: You can install both sets of filters separately, together, or selected set if you want to enable only one.
